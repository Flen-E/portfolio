요리레시피 프로그램 2022.03~06

관련 자료

요리 레시피 검색 편의를 위해
만개의 레시피, 백종원 요리 레시피 url, 쿠팡 재료 링크 등
크롤링을 통하여 DB에 저장 후 재료를 선택하여 레시피 고르기,
각종 요리 레시피 유튜브 시청,
필요한 재료를 검색하여 가격 및 정보와 연결링크 등 다양한 기능을 추가하였습니다.

담당 업무

  • 프로젝트 주제 선정
  • 요리 레시피 크롤링, 쿠팡 요리 재료 크롤링
  • DB 설계 및 관리
  • SceneBuilder를 이용한 front-end 개발

구체적인 구현 기능

      프로젝트 주제 선정
    • 팀원과 각자 주제를 발표하며 투표로 주제 선정
      요리 레시피 크롤링, 쿠팡 요리 재료 크롤링
    • python으로 beautifulsoup를 이용하여 만개의 레시피, 쿠팡 크롤링
    • 마트 위치 검색을 위해 공공데이터 사이트의 api이용
      DB 설계 및 관리
    • mySQL을 이용하여 DB 설계 및 관리
    • 크롤링한 데이터를 DB에 넣어줄 자바프로그램 개발
      SceneBuilder를 이용한 front-end 개발
    • swing을 사용해야하는 제약으로 인해 SceneBuilder를 이용한 fxml파일 제작

사용 기술

  • 개발 언어 : java
  • 개발 툴킷 : java swing
  • 프레임워크 : spring
  • DBMS : MySQL
